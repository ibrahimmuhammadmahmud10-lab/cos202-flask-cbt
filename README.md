
# Mini CBT Engine (Flask)

This project is a simple Computer Based Test system built using Flask.

## Features
- Users answer questions online
- Score is calculated automatically
- Timestamp when test is submitted
- Uses Queue data structure
- Uses Object Oriented Programming

## How to Run

1. Install Python
2. Install Flask

pip install flask

3. Run the app

python app.py

4. Open browser and visit:

http://127.0.0.1:5000
